
# Amazon Product Intelligence Dashboard

# Project Overview

This project analyzes Amazon product data to uncover insights related to:

- Product pricing patterns
- Customer rating behavior
- Product quality consistency
- Customer trust based on reviews
- Premium vs budget product segmentation

The dashboard was built using Tableau Public with interactive visualizations, KPI cards, filters, and story-driven analytics.

# Key Insights

- Premium-priced products generally maintained stronger customer ratings.
- Higher review counts showed stronger customer trust and engagement.
- Certain categories displayed inconsistent product quality patterns.
- Customer sentiment varied across product categories.
